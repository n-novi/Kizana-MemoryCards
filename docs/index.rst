Документация разработчика Kizana MemoryCards
============================================

В этой документации представлен обзор приложения Kizana MemoryCards.

О приложении
============

Kizana MemoryCards - это приложение для запоминания информации с помощью цифровых карточек. Оно использует библиотеку PyQt для создания графического пользовательского интерфейса и SQLite в качестве системы управления реляционными базами данных (СУБД) для хранения и управления данными карточек.

PyQt
----

PyQt - это набор привязок Python для фреймворка приложений Qt, разработанного компанией The Qt Company. Он позволяет программистам на Python создавать настольные приложения с богатым графическим пользовательским интерфейсом (GUI), используя обширный набор инструментов и библиотек Qt.

SQLite
------

SQLite - это легковесный, сервер-менеджер, автономный и открытый SQL-движок баз данных. Это популярный выбор для встроенных приложений баз данных и небольших и средних веб-сайтов, работающих на базе данных. SQLite предоставляет простой и эффективный способ хранения и извлечения структурированных данных без необходимости отдельного серверного процесса.
